# Superstore-Sales-Analysis
This Power BI project employs KPIs to monitor orders, sales, profit, and ship days, complemented by slicers for region (East, South, West, North). It features various chart types to analyze payment mode, orders, segment, month, and ship mode, while maps visualize profit and sales by state.
